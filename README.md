#  Cybersecurity Awareness Chatbot

**PROG5121 - Part 1, 2 & 3 (Final POE)**

A Windows Forms chatbot that educates South African citizens about cybersecurity threats like phishing, password safety, safe browsing, and suspicious links.

---

##  Project Overview

South Africa has seen a significant rise in cyberattacks targeting individuals, businesses, and government institutions (Pieterse, 2021). This chatbot was developed to help educate citizens in a conversational way, making cybersecurity awareness accessible and engaging.

This project was developed in three parts:
- **Part 1:** Console-based chatbot with voice greeting, ASCII art, and colored text
- **Part 2:** Windows Forms GUI with dark theme and chat bubbles
- **Part 3:** Task Manager, Cybersecurity Quiz, Activity Log, and MySQL database integration

---

##  Features

### Part 1 – Console Version
-  Voice greeting (plays on startup)
-  ASCII art logo
-  Personalized conversation using user's name
-  Password safety tips
-  Phishing scam warnings
-  Safe browsing practices
-  Suspicious link detection
-  Colored text and typing effect

### Part 2 – Windows Forms GUI
-  Dark theme GUI with chat bubbles
-  Blue bot messages |  Green user messages
-  Voice greeting (plays after window loads)
-  Quick access buttons for features
-  All Part 1 topics in a clean interface

### Part 3 – Advanced Features
-  **Task Manager** – Add, view, complete, and delete cybersecurity tasks
-  **MySQL Database** – All tasks and logs stored permanently
-  **Cybersecurity Quiz** – 11 multiple-choice questions with feedback and score
-  **Activity Log** – Tracks all user actions and bot responses
-  **NLP Simulation** – Keyword detection for flexible user input
